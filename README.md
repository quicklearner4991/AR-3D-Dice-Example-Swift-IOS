# AR-3D-Dice-Example-Swift-IOS
You can add 3D Dice using ARKit , Roll them and Delete them.
XCODE 12.4 Swift 5

INSTALLATION NOTES:-

1 - Download or clone the repo.

2 - Open project in xcode.

3 - Run the project.
